Flixter App

Java, Movie Database API, Android Async HTTP, Glide

This project was created as the first project in the Android App Development course offered by Codepath. The app was created full through Java and was intended to work as an app that displays a list of current movie showings at the theatres.

The app implements the Movie Database API to display the list of movies at theatres. It also utilizes open-source libraries such as the Android Async HTTP to get HTTP requests asynchronously with JSON parsing and also Glide to load images and cache libraries for Androids.
